# Dead Reckoning

Your GitHub profile as an active archaeological excavation.

Repositories surface as catalogued field specimens, each assessed for material composition, condition rating, and recovery depth. Artifact descriptions are written in the voice of a dry, overly serious field researcher who has never once touched a computer.

Paste any GitHub username into the Open Field Survey at the bottom to generate a field report for any developer's profile.

Built with React, TypeScript, and GSAP.
